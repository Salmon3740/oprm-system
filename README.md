# OPRM - Online Project Resource Management System

## Overview
OPRM is an advanced management platform designed for efficient tracking and allocation of project resources. It leverages modern full-stack technologies and AI integration to provide actionable insights.

## Tech Stack
- Backend: Java 17, Spring Boot, MySQL
- - Frontend: React.js, Tailwind CSS
  - - AI Engine: Python (AI Bridge)
   
    - ## Features
    - - Project Lifecycle Management
      - - Resource Allocation Tracking
        - - AI-based Recommendations
          - - JWT Authentication
